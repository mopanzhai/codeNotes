GIT 笔记

$ git init 新创建仓库
$ git init
$ git add  把文件添加到仓库
$ git add readme.txt
$ git commit 把文件提交到仓库
$ git commit -m"readme file"
-m后面跟说明

$ git status 查看结果,时刻掌握仓库当前状态
$ git status
$ git diff 查看和上次不同地方
$ git diff gitNotes.html
$ git log 查看命令的历史记录
$ git log
$ commit id 版本号
$ git log --pretty=oneline 简介历史记录信息
git reset
返回到之前版本
$ git reset -hard HEAD^
  HEAD^上一版本，HEAD^^上上一版本
$ git reset -hard commit_id
  返回到某个版本

$ cat 
$ cat readme.txt
查看当前版本信息
$ git reflog
$ git reflog  记录每次记录命令
$ git checkout -- readme.txt
把readme.txt文件在工作区的修改全部撤销
撤销暂存区的数据
$ git reset HEAD file
把暂存区的修改撤销掉（unstage），重新放回工作区
$ git rm file
$ git rm test.txt  删除文件
$ ssh-keygen -t rsa -C "youremail@example.com"
